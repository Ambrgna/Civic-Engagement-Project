What is Pokémon?

Pokémon is a turn based RPG video game for the Nintendo handheld consoles. Pokémon has a rock–paper–scissors play style where a "rock" Pokémon can defeat a "scissors" Pokémon but, is weak to a "paper" Pokémon. The Pokémon video game series started about twenty years ago with Pokémon Red and Blue.

These games have a total of 151 unique Pokémon. Now with the newest games Pokémon Ultra Sun and Ultra Moon there is over 800 unique Pokémon.


What makes a Pokémon unique?

There are many factor that make up a single Pokémon. The first is its species the species is the Pokémon. Another factor is its typing. There are 18 types in Pokémon and each Pokémon has a one type and some has a second type. The different types are like rock–paper–scissors each type is weak to some type but, strong against others. Stats are another factor where a Pokémon may be good in one aspect but, bad in another. The last important factor is moves. Each Pokémon can learn a maximum of four moves. Each Pokémon can learn different moves, some Pokémon can learn hundreds of moves and others can learn very few moves. An example of this is Smeargle being able to learn every move and, Ditto can only learn a single move. The last factor is abilities you can learn about them on bulbapedia.

In Pokémon you can use up to six different Pokémon in each battle. This lets you make a balance team that will let you coverage the many different roadblocks that will stand in your way.

This website will help you learn about the different factors and help build a balance Pokémon team.
